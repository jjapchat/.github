# jjapchat(짭챗) 프로젝트

## 1. 소개

- Backend Engineering과 AI Chatbot System에 관련된 기술 스택들을 활용할 수 있는 시스템을 구축하고 개선하는 사이드 프로젝트이다.


## 2. 목표

- Backend Engineering
  - AWS Infra 구축
  - IaC
  - CI/CD (Jenkins?)
  - 대용량 트래픽 생성 및 대응
  - JPA
  - 디자인 패턴
  - Spring Security
 
- AI Chatbot
  - AI System 개요
  - Spring AI
  - Vector DB
  - LangGraph
  - Model 실험 F/W
